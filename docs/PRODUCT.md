# GitPlayground — продукт и learner journey

## Ценность

Платформа для практики Git: теория, квиз и интерактивные задачи в изолированной песочнице с автоматической проверкой.

## Путь ученика

1. **Регистрация / вход** → профиль с баллами и достижениями.
2. **Теория уровня** (`/theory/<level>/`) — контекст перед задачами.
3. **Список задач** (`/tasks/`) — линейная разблокировка внутри уровня.
4. **Плейграунд** (`/playground/<task_id>/`) — терминал + редактор файлов + проверка.
5. **Подсказки** — до 2 за баллы, последовательно.
6. **Квиз** (`/quiz/`) — закрепление терминов, серии и рекорды.
7. **Лидерборд** — рейтинг по баллам профиля.

## Механики прогресса

| Механика | Где |
| --- | --- |
| Баллы за задачу | `Task.points`, начисление при `validate_task` PASSED |
| Списание за подсказку | `HINT_UNLOCK_COSTS` в `learn_ops.py` |
| Достижения | `apps/achievements/services.py`, тост при validate |
| Unlock задач | `can_open_task`, `get_next_unlockable_task_for_user` |
| Ревизии задач | `TaskRevision` + progress |

## Метрики успеха (рефакторинг)

- Coverage ≥ 52% (gate), цель 80%+
- 0 critical security findings в sandbox
- E2E: auth + tasks + playground validate + quiz
